# Support

Welcome! Whether you're setting up Astrokat for the first time, running into an issue, or just curious about what the bot can do, you're in the right place. This document is the central hub for getting help. Refer to the relevant sections below that best fits your situation.

## 📖 Documentation

Start with the **[README](../README.md)**. This covers setup, configuration, and deployment for both Docker and local development.

## 🐛 Found a Bug?

Open a **[Bug Report](https://github.com/mizarc/astrokat/issues/new?template=bug_report.yml)** if you've encountered a reproducible issue.

## 💡 Have a Feature Idea?

Open a **[Feature Request](https://github.com/mizarc/astrokat/issues/new?template=feature_request.yml)** to suggest something new.

## 🤔 Questions & Discussions

For general questions, open-ended ideas, or help getting started, use **[GitHub Discussions](https://github.com/mizarc/astrokat/discussions)** instead of opening an issue.

## 🔒 Security Issues

If you've found a security vulnerability, **do not** open a public issue. See [`SECURITY.md`](../SECURITY.md) for reporting instructions.

## ☕ Thank You

If Astrokat is useful to you, consider **[buying me a coffee](https://buymeacoffee.com/mizarc)** to support development.
